# odin-recipes
Recipe website. First Odin_Project Project 

This is the first project undertaken as part of the Odin Project Foundations course.

The aim is to create a functional - but not pretty - recipe website.

This project will demo the following skills:
- Repository creation and structure in Github
- Use of Git to clone, add, commit and push files: 
- Creation of linked HTML files
- Insertion of imgs, text, lists, headings, and links (relative and absolute)
- task completed using VSCODE
